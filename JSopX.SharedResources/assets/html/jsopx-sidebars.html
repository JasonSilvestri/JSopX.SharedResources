Something to Think About: Headers, Footers, Navbars, Sidebars & Off-Canvas Menus are all awesome ways to connect features and reduce redundant menus. However, take it slow. Carefully explore each section and the source. Done right, it looks epic. Done wrong, like the wrong placement of a component, could induce unwanted results.

JSopX.Sidebars

This example shows how responsive sidebars can work in conjunction with navbars, footers, headers, off-canvas menus, etc. For positioning of navbars, checkout the top and fixed top examples.
